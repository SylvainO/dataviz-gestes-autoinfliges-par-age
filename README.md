# Hospitalisations pour gestes auto-infligés — France (2019–2024)

Visualisation interactive des taux d'hospitalisation pour gestes auto-infligés (tentatives de suicide, automutilations) par tranche d'âge, de 2019 à 2024.

## Source des données

**Santé Publique France** — [Gestes auto-infligés : Hospitalisations (France)](https://odisse.santepubliquefrance.fr/explore/dataset/gestes-auto-infliges-hospitalisations-france/information/)  
Open Data des indicateurs en santé (ODISSE) · Programme de médicalisation des systèmes d'information (PMSI-MCO)
